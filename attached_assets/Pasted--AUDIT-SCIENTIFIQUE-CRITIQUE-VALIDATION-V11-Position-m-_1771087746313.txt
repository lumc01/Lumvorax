# 4. Quad Precision : ce que cela prouve réellement

Norme IEEE 754-2008 :

* double : 53 bits mantisse
* quad : 113 bits mantisse

Quad precision :

* réduit dérive
* augmente stabilité

Mais :

Elle ne crée pas de nouvelle physique.

Si phénomène persiste en 128-bit :

→ cela prouve seulement que ce n’est pas un simple overflow.

# 9. Explication pédagogique (niveau débutant)

## Pourquoi la précision change tout ?

Imaginez tracer une orbite circulaire :

* en 64-bit → petite erreur chaque tour
* en 128-bit → erreur 10¹⁹ fois plus petite

Dans un trou noir :

les équations sont extrêmement sensibles.
Une erreur minuscule peut devenir énorme.

Ce n’est pas que la physique change.
C’est que le calcul devient fragile.
